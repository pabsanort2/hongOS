  • ¿Qué tipo de hongos puede clasificar la aplicación?
    La aplicación puede clasificar los siguientes tipos de hongos: Agaricus, Amanita, boletus, Cortinarus, Entoloma, Hygrocybe, Lactarius, Russula y Suillus. Dentro de éstos grupos, no clasifica ninguna subespecie, por ejemplo no distingue entre Amanita muscaria de la Amanita pantherina, simplemente lo clasifica como Amanita en general.
  • ¿Cuáles son las fuentes de datos del entrenamiento del modelo?
    El set de datos de entrenamiento de la aplicación proviene de un post en kaggle. Se usó este set de datos ya que tenía buena cantidad de imágenes para el entrenamiento además de tener bastantes grupos de clasificación y una estructura de datos bastante bien organizada.
  • ¿Es posible comprobar antes de hacer inferencia si la imagen ha sido usada en el dataset de entrenamiento?
    Sí, esto es posible y se ha implementado como funcionalidad en la aplicación. Se han cargado en la base de datos algunos datos básicos sobre las imágenes usadas para entrenar el modelo, estos datos son tamaño de la imagen, resolución y nombre del archivo. Antes de hacer inferencia se comprueba si coinciden la resolución y el tamaño de la imagen o, en su defecto, si coincide el nombre del archivo. Si alguna de las dos (o las dos) coincide, aparecerá un mensaje de aviso en la clasificación mostrando que es probable que la imagen que ha subido el usuario probablemente se haya usado para entrenar el modelo de clasificación.
